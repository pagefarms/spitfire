# Spitfire Works

One-page Vite + React site for Spitfire Works, an AI-assisted execution team for
small PE firms and portfolio companies with messy operations, underbuilt
systems, and expensive manual work.

## Run locally

```bash
npm install
npm run dev
```

## Validate

```bash
npm run lint
npm run build
```

## Assets

The current build uses the Richmond pixel image at
`src/assets/richmond-pixel.png`. No generated prompt set is part of the active
site direction.
